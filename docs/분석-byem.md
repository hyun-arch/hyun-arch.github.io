# 분석 — byem.vercel.app

> 사장님 요청(2026-07-05): "이 웹사이트 전체 디테일과 기능 하나하나 다 분석하고 구조화해서 내 웹페이지에도 적용해."
> 헤드리스 렌더링으로 DOM·CSS를 뽑아 분석함.

## 1. 무엇인가
퍼포먼스 마케터(문주희/Emily)의 **퍼스널 브랜드 사이트**.
"성과를 한 번 만드는 데서 끝나지 않고, 반복해서 만들어지는 운영 구조를 설계한다"는 컨셉.

## 2. 페이지 구조
- `/` 홈 — 히어로 + 선택 작업 + 서비스 패키지 + WHY ME + 프로세스 + 템플릿 + 문의
- `/projects` 프로젝트 (사례별 상세: `/projects/...`)
- `/services` 서비스 (탭 필터: `?tab=performance|landing|dashboard|ai-consulting`)
- `/templates` 템플릿 (+ 대기자 명단)
- `/insights` 인사이트 (+ `/insights/sessions`, `/insights/skills`)
- `/contact` 문의 (폼)

## 3. 기능/인터랙션
- 프로젝트 사례 카드 (문제 → **결과** 강조박스 → 진행)
- 서비스 패키지 카드 + 탭 필터링 + "시작가" 가격
- 4단계 프로세스 흐름
- 뉴스레터/템플릿 대기자 신청 폼
- 문의 폼 (2영업일 응답)
- 습관 트래커 임베드 (21일 스트릭, 78%)

## 4. 디자인 언어 (핵심 — 우리가 배운 것)
| 요소 | 값 |
|---|---|
| 배경 | 따뜻한 크림 `#faf6ef` / 탄 `#f2eadd`, 섹션마다 교차 |
| 포인트색 | 코랄 `#c2603b` (진하게 `#9c4628`) — 색은 이거 하나만 |
| 먹색 | 따뜻한 near-black `#211c16` (검정 섹션 배경) |
| 헤드라인 | 매우 볼드(800), 강조어만 코랄 |
| 소제목(kicker) | **모노스페이스**(Spline Sans Mono) 대문자 + 자간 넓게 |
| 카드 | radius 18~22px, 옅은 그림자, hover 살짝 뜸 |
| 버튼 | 알약(pill) — 먹색 채움 + 아웃라인 |
| 결과박스 | 코랄 틴트 배경 `#f6e4d9`에 "결과" 라벨 |
| 검정 섹션 | WHY ME — 01~04 넘버 + 모노 번호 |

## 5. 폰트
Pretendard(본문) + Noto Sans KR + Inter + **Spline Sans Mono**(소제목/숫자).

## 6. 배운 것 → 아라미러 적용 원칙
- 내용(마케팅 카피)은 **베끼지 않는다** — [[first-person-not-copied]] 원칙.
- **디자인 언어·컴포넌트 패턴만** 배워서 아라미러 자체 콘텐츠에 입힌다.
