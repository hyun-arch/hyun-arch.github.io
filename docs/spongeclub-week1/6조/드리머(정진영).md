# 드리머(정진영) — 보험 설계 OS — 계약 리포트 자동 정리

- **조**: 6조
- **한 줄**: 통합 계약 리포트 PDF를 올리면 컨설팅 전/후 비교 엑셀의 기존 계약 열을 자동으로 채움

## 문제
담보 누락·순서 보정과 비교 엑셀에 옮겨 적는 반복에 시간 과다

## 아이디어
3층·5단계 루프를 매핑하되 판단(신규 상품)은 남기고 업로드→정리만 자동화, 2건으로 정확도 검증, 애매하면 안 채우고 되묻기

## 태그
#보험자동화 #PDF→엑셀 #판단 보존

## 아람미러 적용
“자동으로 채우는 칸”과 “내가 판단하는 칸”을 명확히 나누고 애매하면 지어내지 말고 되묻기
