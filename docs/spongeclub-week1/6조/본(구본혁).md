# 본(구본혁) — 영감 수집함 — 모일수록 강해지는 OS

- **조**: 6조
- **한 줄**: 이미지·영상·링크·메모를 던지면 자동 태그+아트디렉터 코멘트로 저장, 막힐 때 섞어서 추천

## 문제
영감이 인스타·카톡·유튜브에 흩어져 필요할 때 못 꺼냄

## 아이디어
“수집이 의무면 안 쓴다” 리스크로 캡처 습관부터 검증, 추천은 v1 단순조합→v2 3모드→v3 후킹, tmux 무중단 봇

## 태그
#영감수집 #아이디어추천 #tmux

## 아람미러 적용
캡처→저널에 자동 태그+한 줄 코멘트, “아이디어 줘” 하면 과거 캡처 2~3개를 섞어 새 조합을 제안
