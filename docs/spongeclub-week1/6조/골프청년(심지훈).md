# 골프청년(심지훈) — 1초 가계부 — 월급날 통장 정리 OS

- **조**: 6조
- **한 줄**: 월급 2곳을 넣으면 자동이체·구독·투자를 빼고 옮길 돈·남는 돈을 자동 계산(HTML 한 장)

## 문제
자동이체가 카드구독·통장인출 두 갈래라 월급날마다 옮길 금액을 다시 계산

## 아이디어
자동이체 장부를 한 곳에 모아 카드구독 합계 자동 산출(1단계), localStorage 탭형 계산기, 월별 일괄·구독 알림 확장

## 태그
#가계부 #자동이체정산 #localStorage

## 아람미러 적용
월급날 루틴 카드(통장별 옮길 금액 자동 계산) 위젯 → 반복 재무 정산을 1초 확인으로
