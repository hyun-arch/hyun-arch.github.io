# 다니(송다은) — Dot — 데일리 체크인 기록 파트너

- **조**: 3조
- **한 줄**: 하루 2번(오전 계획·저녁 회고) 체크인으로 운동·돈·생각을 자동 분류 기록

## 문제
운동·생각·돈이 기록될 자리가 없어 전부 잊히고 흘러감

## 아이디어
매일의 점(dot)을 선으로 잇기 — 자연어로 던지면 3카테고리 자동분류 날짜별 MD, 월말 자동 요약 3단계

## 태그
#데일리체크인 #자동분류 #기록습관

## 아람미러 적용
저널에 “오전 계획→저녁 실행 확인” 2회 체크인 루프 → 단순 기록이 아닌 약속 지킴 데이터
