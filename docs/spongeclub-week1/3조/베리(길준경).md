# 베리(길준경) — 베리 리마인더

- **조**: 3조
- **한 줄**: 아이디어·업무지시·할일을 던지면 자동분류 저장하고 하루 2회 다시 알려주는 봇

## 문제
수시로 일이 끊겨 아이디어도 시킨 일도 잊고, 계획 유효기간이 3일을 못 넘김

## 아이디어
코치가 2주 1회 해주던 걸 매일에 심기 — 아이디어/위임/할일 자동분류, 9시·16시 리마인드, 음성도 오프라인 STT

## 태그
#리마인더 #위임추적 #음성입력

## 아람미러 적용
나이틀리 합성에 “미완료는 답할 때까지 다음날 아침 리포트에 재등장” → 잊힘을 구조적으로 차단
