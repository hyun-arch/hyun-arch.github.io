# 제인(최지연) — 데일리→월간 재사용 마케팅 리포트 OS

- **조**: 3조
- **한 줄**: 매체+GA를 한 행에 담는 통합 로데이터로 재설계해 월간 리뷰 취합을 없앰

## 문제
데일리에 숫자만 수기로 들어가 월간마다 로데이터를 처음부터 다시 취합

## 아이디어
행이 아니라 열을 제대로 — 날짜/월/매체/지표를 한 뼈대에, 월 칼럼 자동추출로 피벗 한 번에 월간 리뷰 끝

## 태그
#리포트자동화 #데이터설계 #마케팅

## 아람미러 적용
