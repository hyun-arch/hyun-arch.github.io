# 양세(박세현) — 영상 아카이브+제작+편집 자동화 OS 3종

- **조**: 5조
- **한 줄**: 영상 링크 4줄 카드 아카이빙, 슬랙 한 줄 숏폼 제작, 매니페스트 JSON 기반 편집 자동화

## 문제
찾아본 영상이 휘발돼 반복 검색하고 기획·생성·편집을 매번 손으로

## 아이디어
①링크→카드 저장+냉정 평가+사용 추적 ②슬랙 지시→콘티·대본·ffmpeg 반자동 ③쇼츠=매니페스트 JSON 한 장으로 멀티 에이전트 분담

## 태그
#영상아카이브 #숏폼 #멀티에이전트

## 아람미러 적용
빌드로그를 “매니페스트 한 장=콘텐츠 한 편” → 글→발표자료→숏폼까지 같은 재료로 자동 재현
