# 챈(이채은) — 매월 반복 데이터 점검 자동화 검증 앱

- **조**: 5조
- **한 줄**: 엑셀을 넣으면 규칙 기반 자동 점검+담당자 통보용 보고문까지, 브라우저 로컬 전용

## 문제
매달 반복하는 데이터 점검이 시간을 가장 많이 잡아먹는 수작업

## 아이디어
업로드→규칙 검사→중요도 2단계 건별 결과+보고문 자동 생성, 개인정보라 클라우드로 안 보내고 브라우저 안에서만

## 태그
#데이터검증 #로컬처리 #보안

## 아람미러 적용
민감 데이터는 “브라우저/로컬에서만 처리, 공개층엔 승격본만” 원칙을 명시적 레이어로
