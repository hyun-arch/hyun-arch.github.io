# 차지원 — 장기렌트 DB 납품 자동화 OS

- **조**: 5조
- **한 줄**: 구글시트↔엑셀 복붙 납품 전 과정을 Apps Script 무인 파이프라인으로

## 문제
수집→취합→중복검사→분배→보정→납품→이력을 매일 전부 손복붙

## 아이디어
시간 트리거로 취합→중복제거→COUNTIFS 검사→우선순위 분배→광고주별 xlsx 자동 생성, 설정은 하드코딩 없이 시트 데이터로

## 태그
#AppsScript #무인파이프라인 #설정의 데이터화

## 아람미러 적용
“규칙은 코드에 박지 말고 데이터(MD/JSON)로 분리” → 카테고리·루틴 시간을 파일 수정만으로 변경
