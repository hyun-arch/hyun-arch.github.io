# 그레이(김수현) — S box — 저장에서 실행으로

- **조**: 5조
- **한 줄**: 던지면 자동 분류·저장하고 토요일마다 하나를 꺼내 실행하게 하는 저장→실행→재개 OS

## 문제
저장만 하고 실행 안 하며, 시작해도 흐름 끊기면 재개 못함

## 아이디어
배우기/해보기/챙기기/업무로 MD 저장·“흐르는 것/고이는 것” 구분, 토요일 10시 추천+캘린더 알림으로 재개

## 태그
#자동분류 #실행재개 #대시보드

## 아람미러 적용
“저장→실행” 재개 루프 → 주말마다 미실행 1개를 골라 “이번 주 이거 하나만” 텔레그램 추천
