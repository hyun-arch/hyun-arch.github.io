# 최강훈 — 정리함 OS — 넣기만 되던 걸 꺼내주기

- **조**: 5조
- **한 줄**: 수집함에 사진·메모를 넣고 “정리해줘” 하면 할일/아이디어/참고자료로 분류해 표로 꺼냄

## 문제
생각·할일·자료가 흩어져 “넣기만 되고 꺼내기가 안 되는” 상태

## 아이디어
수집함→분류→db 표 저장→보관 이동 파이프라인, 강의 필기 4장이 17건 실행 가능한 표로

## 태그
#수집함 #자동분류 #꺼내기

## 아람미러 적용
저널→위키 승격 사이에 “수집함→표로 꺼내기” 중간 단계 → 던진 것들이 표로 정리돼 승격 판단이 쉬움
