# 하니(우동한) — 삼보수산 홍합 컨디션 아카이브 OS

- **조**: 4조
- **한 줄**: 매일 찍는 홍합 사진을 던지면 날짜·알크기·색·코멘트가 MD로 자동 저장(5층 기획)

## 문제
1년 반치 홍합 사진이 창고에만 쌓이고 6개 플랫폼 반복 업로드 부담

## 아이디어
저장→스킬화(문구+6플랫폼)→지식(연간 품질지도 위키)→자산화(글감)→사고파트너(“작년보다 알이 크네요”)

## 태그
#데이터자산화 #사진아카이브 #연간패턴

## 아람미러 적용
매일 캡처에서 “작년 오늘의 나”와 비교해주는 사고 파트너 브리핑 → 반복 기록을 자산으로 되살리기
