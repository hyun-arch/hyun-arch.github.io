# 이니(죠죠) — 나루(iny-OS) — 배움을 자산으로 건네는 나루터

- **조**: 4조
- **한 줄**: 생각을 던지면 콘텐츠·포트폴리오 자산이 되는 개인 OS(웹 대시보드+봇+자동 동기화)

## 문제
강의·독서로 배운 게 휘발되고 콘텐츠 재료로 정리하는 단계가 비어 있음

## 아이디어
던지기→여울(발산)→너울(실행)→웹허브→링크드인 재발행, Self/External 분리, board.json 커밋훅 자동 갱신

## 태그
#지식자산화 #파이프라인 #자동동기화

## 아람미러 적용
“여울/너울”처럼 발산(사색)과 실행(초안) 채널 분리 + 진행현황이 커밋훅으로 자동 갱신되는 대시보드
