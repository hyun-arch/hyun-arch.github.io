# 달빛그린(윤지윤) — 피드백 정리 OS — 설득에 쓰던 시간을 디자인에

- **조**: 1조
- **한 줄**: 뒤죽박죽 디자인 피드백을 붙여넣으면 UX·데이터 기준으로 자동 분류하고 처리 결정서까지 생성

## 문제
비전문가 피드백까지 섞여 매번 손으로 판단·근거 만들어 설득하느라 프로젝트가 산으로 감

## 아이디어
출처·분류·UX부합·데이터근거·권장처리 5칸 표로 분류 → 채택/기각 근거 붙은 결정서 자동 생성
