# 띵크(이예성) — 로그인 없이 켜면 오늘 할 일이 다 보이는 데스크탑 캘린더 OS

- **조**: 1조
- **한 줄**: 폼 제출·미팅·Zoom 요약을 로그인 없이 한 화면에 모아 보여주기

## 문제
확인하려면 매번 회사 사이트 로그인 후 여러 단계를 거쳐야 함

## 아이디어
진짜 마찰은 “로그인해 들어가는 것 자체” → 폼목록 → 미팅+Zoom요약 → 완성형 캘린더 3단계

## 태그
#캘린더 #무로그인 #대시보드

## 아람미러 적용
첫 화면을 “켜면 오늘 일정·할일·리포트가 다 보이는” 무로그인 원스크린 대시보드로 유지·강화
