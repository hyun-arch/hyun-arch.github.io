# 로밍(박흥순) — Biz 집사 — 여러 사업체 챙길 일 운영 OS

- **조**: 1조
- **한 줄**: 여러 법인의 세금일·주총·중임 기한을 봇이 문서흡수(OCR)·기한계산·하루 2회 브리핑

## 문제
여러 법인의 납부일·갱신·중임·주총이 흩어져 매번 세무사·검색 총동원

## 아이디어
봇1개+법인별 그룹방, PDF/이미지 던지면 OCR 흡수하고 상법 근거와 함께 답변, 아침·오후 브리핑+D-3 알림

## 태그
#법인운영 #텔레그램봇 #OCR

## 아람미러 적용
아침 리포트에 “완료/아직 버튼 + 미완료 이월 + D-3 기한 알림” 패턴 이식
