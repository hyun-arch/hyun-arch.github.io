Calendar · 일정 · 예약 · 미팅 OS

Luxe Calendar — 일정·예약·미팅 조율 앱

일정 관리에서 끝나지 않고, 예약 링크로 남이 내 빈 시간을 잡고, 미팅 시간을 자동으로 조율하고, 구글 캘린더와 양방향 동기화까지 하는 개인 스케줄 OS.

스택 Next.js 14 · Prisma(Supabase) · NextAuth(Google) · Electron

연동 구글 캘린더 — 여기서 바로

사장님 구글 캘린더(본인 + 볼로냐·전시미팅·회의실·AIIA·Seongnam 프로젝트 캘린더)를 모두 합쳐 이 화면에 박아뒀습니다. 아래에 오늘 일정이 그대로 보이면 연동된 겁니다.

🔑 일정이 안 보이면 이 브라우저에 hyun@aramhuvis.com 구글 계정으로 로그인돼 있는지 확인하세요 (비공개 캘린더라 로그인해야 사장님 일정이 뜹니다). → 구글 캘린더 전체 열기

기능 다섯 가지

앱의 상단 탭이 곧 기능이다. 일정만 보는 달력이 아니라 예약·미팅까지 조율한다.

01

일정 (Calendar)

월·주·일 보기로 일정을 관리. 구글 캘린더와 동기화되어 한 곳에서 본다.

02

미팅 조율 (Meeting)

상대에게 후보 시간을 제안 → 상대가 고르면 확정. 서로 캘린더 안 맞춰도 됨.

03

예약 링크 (Links)

내 예약 페이지 링크(+QR)를 공유하면, 상대가 내 빈 슬롯 중 골라 예약한다.

04

알림 (Alerts)

웹 푸시 + 이메일 + 슬랙으로 일정·예약 리마인더를 자동 발송.

05

설정 (Settings)

구글 연동 상태, 알림 채널, 계정 설정을 한 곳에서.

연동 뒤에서 자동으로

화면 뒤에서 구글·알림·메일이 자동으로 돈다.

구글 캘린더 동기화 googleapis로 양방향 동기화 + iCal 내보내기 (calendar/sync · calendar/ical)
자동 리마인더 cron이 다가온 일정·예약을 감지해 미리 알림 (cron/reminders)
웹 푸시 알림 web-push(VAPID)로 브라우저·PC에 바로 알림 (notifications/subscribe)
이메일 · 슬랙 nodemailer(SMTP) 메일 + 슬랙 웹훅으로 확정·리마인더 발송

실행 두 가지 방식

셸 봇과 같은 구조 — 웹(개발 서버)과 PC 앱(Electron), 둘 다 한 레포에 들어있다. 실제 코드: Project/Calender.

방식 A

웹 · 개발 서버

가장 간단

레포에서 npm run dev → 브라우저로 http://localhost:3000 접속

  • 실행 위치내 PC(개발 서버) · 팀에 열려면 배포(Vercel 등) 필요
  • 필요한 것Node · .env(구글 OAuth·DB·SMTP) 설정
  • 명령·파일npm run dev
방식 B

PC 앱 · Electron

데스크톱

독립 실행 데스크톱 앱으로 빌드해 아이콘 눌러 실행 (Luxe Calendar.exe)

  • 실행 위치내 PC · 브라우저 없이 창으로
  • 필요한 것npm run electron:build (win: nsis 설치본)
  • 명령·파일electron/main.js · npm run electron:start

설정 처음 한 번 (사장님 몫)

구글·DB·메일 키는 민감값이라 코드로 못 넣는다. 아래만 한 번 채우면 굴러간다.

  1. 구글 OAuth 클라이언트 생성 → GOOGLE_CLIENT_ID / SECRET
  2. Supabase PostgreSQL → DATABASE_URL (prisma migrate)
  3. NEXTAUTH_SECRET · SMTP(메일) · VAPID(웹푸시) 키 발급
  4. .env 에 채우고 npm run dev 로 로그인 → 구글 연동

🟡 이 앱은 서버·로그인·DB가 필요한 동적 앱이라, 정적 라이브(GitHub Pages)엔 통째로 올리지 않는다. 위 방식으로 내 PC에서 돌린다.
