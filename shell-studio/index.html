04 두 가지 방식 — 웹(클라우드) · PC 앱

셸 분기를 두 가지로 다 만들어 실제로 돌려봤습니다. 웹(클라우드)과 내 PC에서 도는 앱, 둘 다 한 레포에 들어있어요. 원하는 걸 쓰면 됩니다.

방식 A

웹 · 클라우드 리마인더

추천

GitHub Actions가 매일 00:01 KST에 reminder.py를 돌려, 오늘 대상 + 바로 쓸 명령어를 내 Slack으로 DM

  • 자동화대상 계산·알림 자동 · 마지막 전송 1탭만 나
  • 실행 위치GitHub 클라우드 — 내 PC 꺼져 있어도 됨
  • 필요한 것표준 라이브러리만(설치 X) · Slack 봇 토큰
  • 안정성매우 높음
  • 파일reminder.py · .github/workflows/daily.yml
방식 B

PC 앱 · 브라우저 완전자동

0탭

Windows 작업 스케줄러가 매일 run_auto.bat 호출 → auto_send.py(Playwright)가 Slack에서 대신 전송

  • 자동화0탭 완전 자동
  • 실행 위치내 PC — 그 시각에 켜져 있어야 함
  • 필요한 것Python + Playwright 설치
  • 안정성Slack 화면 바뀌면 보정 필요
  • 파일auto_send.py · run_auto.bat

05 사용 매뉴얼 — 1~6조 누구나 쓰게

봇 실제 코드는 Project/자동 셸 선물 봇. 조·명단만 바꾸면 다른 조도 그대로 씁니다.

공통 준비 (먼저 이것부터)

  1. 이 레포를 복제(clone)한다
  2. config.json — team을 내 조로, mode를 rotate_all(전체 순환) 또는 favorites로
  3. data/members.json — 우리 조 명단(닉네임·이름·slack_id) 채우기
  4. 미리보기: 터미널에서 python rotation.py → 오늘 누가 대상인지 확인

방식 A방식 A · 웹(클라우드) 설치

  1. Slack 앱 만들고 봇 토큰 발급 (권한: chat:write, im:write)
  2. 레포 Settings → Secrets and variables → Actions → New secret 에 SLACK_BOT_TOKEN 저장
  3. Actions 탭에서 "매일 셀 리마인더" 워크플로우 켜기
  4. 끝 — 매일 00:01 KST에 오늘 대상 DM이 온다 (내 PC 꺼져 있어도 됨)

방식 B방식 B · PC 앱(완전자동) 설치

  1. Python 설치 → pip install playwright && playwright install
  2. run_auto.bat의 Python 경로를 내 PC 경로로 맞추기
  3. Windows 작업 스케줄러에 매일 원하는 시각으로 run_auto.bat 등록
  4. 끝 — PC가 켜져 있으면 그 시각에 Slack에서 자동 전송(0탭). 로그는 run.log

06 수집봇 — 셸 버는 쪽

선물봇이 셸을 쓰는 쪽이라면, 수집봇은 셸을 버는 쪽입니다. 매일 사라지는 셸을 벌 기회를 놓치지 않게 도와줍니다.

💡 가장 큰 기회는 공유회 개최입니다. 매일 받는 셸 1개보다, 개최 1회(7월 한정 20개)가 훨씬 큽니다.

①

셸 기회 감지 알림

collector.py

공지·공유회·5조 채널을 30분마다 감시해서, 셸 벌 기회가 올라오면 내 DM으로 즉시 알림.

감지 선착순이벤트셸 지급공유회 개최·모집인증하면

일상적인 셸 주고받기 봇 메시지는 잡음이라 자동 제외

python collector.py새 기회 감지 → DM
python collector.py --dry-run감지만, DM 안 보냄
python collector.py --reset기준점 초기화

⏱ 자동 실행 · 작업 스케줄러 SpongeClub-Shell-Collector (30분 간격)

②

SNS 인증 도우미

sns_verify.py

실제로 올린 SNS 게시물 링크를 주면 /sns인증 제출을 자동화. 관리자 승인 후 +1셸.

python sns_verify.py https://www.instagram.com/p/XX␣/SNS 링크로 인증 제출

⚠️ 게시물 없이 자동으로 인증을 쏘는 기능은 일부러 넣지 않았습니다 (실제 활동만 인증).
